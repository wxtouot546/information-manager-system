lucene是什么

Lucene是一款开源的、高性能的、可拓展的信息检索Java工具库(也就是说lucene是提供搜索功能的Java jar包)。具体是下图的东西，也可以下载下来看一眼是什么。(下载链接)。将lucene引入Java项目中，你可以利用它做出类似百度、google提供的搜索功能。

什么时候应该考虑使用搜索引擎lucene

第一点：搜索功能

想为应用添加类似百度的搜索功能

第二点: 搜索速度

当mysql、oracle数据库搜索的速度太慢时，举一个例子，假设你有一亿篇文章，使用数据库提供的like ‘%关键词%’，将会耗时几分钟才能查找出结果。而搜索引擎则能做到几十毫秒查询到结果。搜索引擎之所以速度比数据库快的原因是，它的底层是倒排索引。

第三点：搜索质量

mysql、oracle数据库搜索的质量太差，想让搜索结果中与搜索关键词最相关的排在前面返回，为搜索结果做一个相关度排序的功能。

lucene、mysql、oracle的区别？

lucene是搜索引擎，mysql、oracle是关系型数据库，问lucene与mysql的区别，不如说是搜索引擎与关系型数据库的区别

搜索引擎 关系型数据库
搜索速度 亿级数据量，可以保持在几十毫秒内 随着数据量上升，如果性能曲线下降
搜索质量 可以为搜索结果自定义相关度排序，让搜索结果中与搜索词相关度较高的词排前面返回 搜索质量较差，数据库一般不关注搜索质量，它着重在检索出来。一般，数据库提供按照数字大小，字符进行排序的功能。
维护数据与数据的关系的能力 弱 专业
提供数据维护的事务功能 没有 专业

File对象

File对象_编辑器使用教程

在计算机系统中，文件是非常重要的存储方式。Java的标准库java.io提供了File对象来操作文件和目录。

要构造一个File对象，需要传入文件路径：

import java.io.*;
public class Main {
    public static void main(String[] args) {
        File f = new File("C:Windowsnotepad.exe");
        System.out.println(f);
    }
}
